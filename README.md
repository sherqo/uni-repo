# CAIE 23P · Junior Repo

Voluntary student collaboration hub for CAIE 23P juniors at Ain Shams University.

## Stack

- [Astro](https://astro.build) + [Tailwind CSS v4](https://tailwindcss.com)
- Package manager: [Bun](https://bun.sh)

## Dev

```sh
bun install
bun run dev
```

## Deploy

Hosted on [Netlify](https://caie23p.netlify.app). Pushes to `main` auto-deploy.
